# generateSitemapTask

Generate a `sitemap.xml` for search engine indexing.

---

## When to Use

✅ Use `generateSitemapTask` when you want:
- Better search engine discoverability
- Automatic sitemap from generated pages

This task should run **last** (or near-last) to capture all generated pages.

---

## Quick Start

```js
generateSitemapTask({
  outDir: 'public',
  baseUrl: 'https://example.com',
})
```

**Output:** `public/sitemap.xml`

---

## Configuration

| Option | Type | Required | Description |
|--------|------|----------|-------------|
| `outDir` | string | ✅ | Directory containing generated pages |
| `baseUrl` | string | ✅ | Site root URL (absolute) |

---

## What's Included

The sitemap automatically includes:
- All `.html` files in the output directory
- Pages generated by `generatePagesTask`
- Pages from `generateItemsTask`
- Paginated pages from `generatePaginatedItemsTask`

---

## Pipeline Position

Place after all page-generating tasks:

```js
export default [
  prepareOutputTask({ /* ... */ }),
  generateItemsTask({ /* ... */ }),
  generatePagesTask({ /* ... */ }),
  generateFeedTask({ /* ... */ }),
  generateSitemapTask({ outDir: 'public', baseUrl: 'https://example.com' }),  // ← Last
  copyStaticTask({ /* ... */ }),  // Assets don't affect sitemap
];
```
